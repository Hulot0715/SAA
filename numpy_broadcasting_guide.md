# NumPy 广播机制（Broadcasting）速查手册

---

## 一、什么是广播？

广播是 NumPy 在形状不完全相同的数组之间执行运算时，**自动扩展维度**的机制。
它让你无需手动复制数据，就能写出简洁高效的向量化代码。

---

## 二、广播的三条规则

从**右侧**对齐各数组的维度，逐轴检查：

| 规则 | 说明 |
|:----:|:----|
| ① | 两轴大小相同 → 正常运算 |
| ② | 其中一轴大小为 1 → 沿该轴复制扩展，匹配另一方 |
| ③ | 两轴大小既不同又都不为 1 → **报错** |

> **核心口诀：大小为 1 的轴可以被广播，其余必须相等。**

---

## 三、np.newaxis 的作用

`np.newaxis`（等价于 `None`）在指定位置插入一个大小为 **1** 的新轴，
为后续广播创造条件。

```python
a = np.array([1, 2, 3])   # shape (3,)

a[:, np.newaxis]          # shape (3, 1)  在末尾插入新轴
a[np.newaxis, :]          # shape (1, 3)  在开头插入新轴
```

## 四、经典例子：外积（Outer Product）

### 目标
计算向量 `[1,2,3]` 和 `[10,20,30]` 所有元素两两相乘，得到 3×3 矩阵。

### 代码
```python
a = np.array([1, 2, 3])          # shape (3,)
b = np.array([10, 20, 30])       # shape (3,)

result = a[:, np.newaxis] * b    # (3,1) * (3,) -> (3,1) * (1,3) -> (3,3)
```

### 广播可视化

```
a[:, np.newaxis]  形状 (3,1)     b  形状 (1,3)（自动在左侧补1）

  ┌────┐                        ┌────┬────┬────┐
  │  1 │                        │ 10 │ 20 │ 30 │
  ├────┤                        └────┴────┴────┘
  │  2 │
  ├────┤
  │  3 │
  └────┘

  列方向复制 3 份 ──→              行方向复制 3 份 ──→

  ┌────┬────┬────┐              ┌────┬────┬────┐
  │  1 │  1 │  1 │              │ 10 │ 20 │ 30 │
  ├────┼────┼────┤   相乘        ├────┼────┼────┤
  │  2 │  2 │  2 │  ──────→    │ 10 │ 20 │ 30 │
  ├────┼────┼────┤              ├────┼────┼────┤
  │  3 │  3 │  3 │              │ 10 │ 20 │ 30 │
  └────┴────┴────┘              └────┴────┴────┘

  结果 (3,3)：
  ┌────┬────┬────┐
  │ 10 │ 20 │ 30 │
  ├────┼────┼────┤
  │ 20 │ 40 │ 60 │
  ├────┼────┼────┤
  │ 30 │ 60 │ 90 │
  └────┴────┴────┘
```

## 五、进阶例子：TSP 城市距离矩阵（三维广播）

### 背景
有 N=5 座城市，每座城市有坐标 `[x, y]`，需要一次性算出所有城市对的坐标差。

### 代码
```python
cities = np.array(...)            # shape (5, 2)

diff = cities[:, np.newaxis, :] - cities[np.newaxis, :, :]
#      (5, 1, 2)                -  (1, 5, 2)
#      广播后均变为 (5, 5, 2)

dist = np.sqrt((diff ** 2).sum(axis=2))   # (5, 5, 2) -> (5, 5)
```

### 插入新轴后的形状可视化

```
cities[:, np.newaxis, :]   形状 (5,1,2)     →  可视为"列向量"，每格是 [x,y]

  ┌───┐
  │ A │   城市0
  │ B │   城市1
  │ C │   城市2
  │ D │   城市3
  │ E │   城市4
  └───┘
   1列

cities[np.newaxis, :, :]   形状 (1,5,2)     →  可视为"行向量"，每格是 [x,y]

  ┌───┬───┬───┬───┬───┐
  │ A │ B │ C │ D │ E │
  └───┴───┴───┴───┴───┘
         5列
```

### 广播相减结果（5×5 矩阵，每格存坐标差向量）

```
         A        B        C        D        E      ← 行向量沿行方向扩展
    ┌────────┬────────┬────────┬────────┬────────┐
 A  │ A - A  │ A - B  │ A - C  │ A - D  │ A - E  │
    ├────────┼────────┼────────┼────────┼────────┤
 B  │ B - A  │ B - B  │ B - C  │ B - D  │ B - E  │
    ├────────┼────────┼────────┼────────┼────────┤
 C  │ C - A  │ C - B  │ C - C  │ C - D  │ C - E  │
    ├────────┼────────┼────────┼────────┼────────┤
 D  │ D - A  │ D - B  │ D - C  │ D - D  │ D - E  │
    ├────────┼────────┼────────┼────────┼────────┤
 E  │ E - A  │ E - B  │ E - C  │ E - D  │ E - E  │
    └────────┴────────┴────────┴────────┴────────┘
 ↑
列向量沿列方向扩展

每格存储的是 [xi - xj, yi - yj]，即二维坐标差向量。
对角线（A-A, B-B...）= [0, 0]，开根后距离为 0。
```

### 最后一步：压缩第三维

```
(5, 5, 2)  --[平方和，axis=2]-->  (5, 5)  --[开根]-->  (5, 5) 距离矩阵

每格 [dx, dy]  →  sqrt(dx²+dy²)  →  欧氏距离标量
```

## 六、常见广播形状速查

| 操作 | 数组A形状 | 数组B形状 | 结果形状 | 典型用途 |
|:----:|:--------:|:--------:|:--------:|:-------:|
| 标量运算 | `(N,)` | `()` | `(N,)` | 全体元素加/乘常数 |
| 行广播 | `(M, N)` | `(N,)` | `(M, N)` | 每行加同一向量 |
| 列广播 | `(M, N)` | `(M, 1)` | `(M, N)` | 每列加同一向量 |
| 外积 | `(M, 1)` | `(1, N)` | `(M, N)` | 两向量所有元素两两运算 |
| 三维外积 | `(N, 1, K)` | `(1, N, K)` | `(N, N, K)` | 城市对坐标差（本例） |

## 七、常见错误

```python
a = np.array([[1,2,3],[4,5,6]])   # shape (2, 3)
b = np.array([1, 2])              # shape (2,)

a + b   # ❌ 报错！右对齐后：(2,3) vs (2,)，末轴 3≠2 且都不为1

# 修正：插入新轴使末轴变为1，再广播
a + b[:, np.newaxis]              # ✅ (2,3) + (2,1) -> (2,3)
```

## 八、判断是否可广播的步骤

```
1. 从右侧对齐两数组的形状，不足的位置在左侧补 1
2. 逐轴检查：相等 ✅  |  其中一方为 1 ✅（可扩展）  |  其余情况 ❌（报错）
3. 结果形状：每轴取两者中的较大值
```

**示例：**
```
A: (   5, 1, 2)
B: (1, 1, 3, 1)   ← 左侧自动补1后
─────────────────
   (1, 5, 3, 2)   ← 每轴取较大值，均满足规则 ✅
```
